# E2EE Encrypted Chat

# Prerequisites

	- NodeJS (Preferred Version v17.9.0)
	- npm

# Setup project dependencies
	- run command
		$ npm i

# Start server
	- run command
		$ npm start
	- Server will run at http://localhost:3000

# Using the application
	- Open http://localhost:3000/ in browser
	- Login using a google account
